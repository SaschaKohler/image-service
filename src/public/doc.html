Überblick

Der HTML-to-Image Service bietet eine elegante Lösung zur Konvertierung von HTML-Inhalten in hochwertige Bilder.

Features

  • HTML zu Bild Konvertierung
  • Template-System
  • Multiple Formate

Vorteile

  • Einfache Integration
  • Hohe Performance
  • Flexible Anpassung

API-Dokumentation

Bild generieren

POST /v1/image
Content-Type: application/json

{
  "html": "
Hallo Welt
",
  "css": "div { color: blue; }",
  "template_data": {
    "title": "Mein Titel"
  }
}

Parameter

  • html: HTML-Content
  • css: CSS-Styles
  • template_data: Variablen

Response

  • url: Bild-URL
  • image_id: Bild-ID

Fehlerbehandlung

200 Erfolg
201 Erstellt
400 Ungültige Anfrage
401 Nicht autorisiert
